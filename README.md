# Welcome to Semantix

**Semantix** is an AI-powered data intelligence platform that lets your team query databases, explore data, and build dashboards using natural language — no SQL expertise required.

## What can you do with Semantix?

- 💬 **Ask questions in plain language** — Semantix translates them into SQL and returns results instantly
- 📊 **Build dashboards** — drag-and-drop charts, scorecards, and tables
- 🔌 **Connect any database** — PostgreSQL, MySQL, BigQuery, Snowflake, ClickHouse, Google Sheets, and more
- 🤖 **Advanced analytics** — Cohort, RFM, Funnel, Growth, and Vintage analysis out of the box
- 🔔 **Automated alerts & reports** — get notified on Telegram, Zalo, or Microsoft Teams
- 🔑 **Public API** — embed dashboards or query data from your own apps
