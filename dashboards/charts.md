# Charts & Widget Configuration

## Configuring a Widget

Each widget on a dashboard can be configured independently:

1. Hover over a widget → click the **⚙️ settings icon**
2. Adjust the query, chart type, title, or display options
3. Click **Save**

## Common Settings

| Setting | Description |
|---------|-------------|
| **Title** | Widget heading shown on the dashboard |
| **Chart Type** | Visual representation type |
| **X Axis** | Column for the horizontal axis |
| **Y Axis** | Column(s) for the vertical axis / value |
| **Color By** | Column used to differentiate series |
| **Aggregation** | SUM, AVG, COUNT, MIN, MAX |
| **Sort** | Sort by axis value, ascending or descending |
| **Limit** | Max number of rows/bars to display |

## Scorecard Widget

Scorecards display a single metric with optional comparison:

| Setting | Description |
|---------|-------------|
| **Value** | The main metric |
| **Comparison Period** | Previous period to compare against |
| **Format** | Number, currency, percentage |
| **Prefix / Suffix** | e.g. `$`, `%`, `K` |

## Table Widget

| Setting | Description |
|---------|-------------|
| **Columns** | Which columns to show and their order |
| **Column Width** | Fixed or auto width |
| **Pagination** | Rows per page |
| **Searchable** | Enable in-table search |

## Resizing & Reordering

On the dashboard edit view:
- **Drag** the widget header to reorder
- **Drag** the bottom-right corner to resize
- Widgets snap to a grid layout automatically
