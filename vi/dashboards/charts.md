# Biểu Đồ & Cấu Hình Widget

## Cấu Hình Một Widget

Mỗi widget trên dashboard có thể được cấu hình độc lập:

1. Di chuột qua một widget → nhấn vào **biểu tượng cài đặt ⚙️ (settings icon)**
2. Điều chỉnh truy vấn, loại biểu đồ, tiêu đề hoặc các tùy chọn hiển thị
3. Nhấn **Save**

## Các Cài Đặt Chung (Common Settings)

| Cài Đặt | Mô Tả |
|---------|-------------|
| **Title** | Tiêu đề widget hiển thị trên dashboard |
| **Chart Type** | Loại biểu đồ hiển thị |
| **X Axis** | Cột dành cho trục ngang |
| **Y Axis** | (Các) Cột dành cho trục dọc / giá trị |
| **Color By** | Cột dùng để phân biệt các chuỗi dữ liệu (series) |
| **Aggregation** | SUM, AVG, COUNT, MIN, MAX |
| **Sort** | Sắp xếp theo giá trị trục, tăng dần hoặc giảm dần |
| **Limit** | Số lượng dòng/cột tối đa hiển thị |

## Scorecard Widget

Scorecards hiển thị một metric đơn lẻ kèm theo tùy chọn so sánh:

| Cài Đặt | Mô Tả |
|---------|-------------|
| **Value** | Metric chính |
| **Comparison Period** | Khoảng thời gian trước đó để so sánh |
| **Format** | Số, tiền tệ, tỷ lệ phần trăm |
| **Prefix / Suffix** | Tiền tố/hậu tố, ví dụ: `$`, `%`, `K` |

## Table Widget

| Cài Đặt | Mô Tả |
|---------|-------------|
| **Columns** | Chọn các cột để hiển thị và thứ tự của chúng |
| **Column Width** | Chiều rộng cố định hoặc tự động |
| **Pagination** | Số dòng trên mỗi trang |
| **Searchable** | Bật tính năng tìm kiếm trong bảng |

## Thay Đổi Kích Thước & Sắp Xếp (Resizing & Reordering)

Trên giao diện chỉnh sửa dashboard:
- **Kéo (Drag)** thanh tiêu đề widget để sắp xếp lại
- **Kéo (Drag)** góc dưới cùng bên phải để thay đổi kích thước
- Các widget sẽ tự động bám vào bố cục dạng lưới (grid layout)
